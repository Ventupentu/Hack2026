\section{Detalles de Implementacion}

\subsection{Configuracion y entrypoints}

El proyecto usa Hydra~\cite{yadan2019hydra} con dataclasses tipadas (\texttt{src/config.py}). Entrypoints operativos:
\begin{itemize}
  \item \texttt{python -m src.train}: entrenamiento OpenCLIP.
  \item \texttt{python -m src.infer}: inferencia principal con salida en directorio Hydra.
  \item \texttt{python -m src.new\_infer}: inferencia per-crop con TTA y FAISS opcional.
  \item \texttt{python -m src.infer\_phase1}: inferencia por seccion + zero-shot.
  \item \texttt{python -m src.infer\_top1}: inferencia top-1 por crop.
  \item \texttt{python -m src.infer\_openclip}: inferencia simple por CLI.
\end{itemize}
