\section{Detalles de Implementacion}

\subsection{Configuracion y entrypoints}

El proyecto usa Hydra~\cite{yadan2019hydra} con dataclasses tipadas (\texttt{src/config.py}). Entrypoints principales:
\begin{itemize}
  \item \texttt{python -m src.train}
  \item \texttt{python -m src.infer}
  \item \texttt{python -m src.new\_infer}
  \item \texttt{python -m src.infer\_phase1}
  \item \texttt{python -m src.infer\_top1}
  \item \texttt{python -m src.infer\_openclip}
\end{itemize}
