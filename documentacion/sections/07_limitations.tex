\section{Limitaciones y Riesgos}

\begin{enumerate}
  \item Dependencia de pesos/modelos externos (OpenCLIP, YOLO).
  \item Coste computacional adicional en variantes con deteccion + rerank.
  \item Dependencias opcionales (FAISS/Ultralytics) fuera del set base de \texttt{requirements.txt}.
  \item Cobertura de tests automatizados todavia limitada.
\end{enumerate}
