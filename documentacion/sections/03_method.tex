\section{Metodologia}

\subsection{Formulacion}

Para cada bundle $b$, el sistema rankea candidatos de producto $\{p_i\}_{i=1}^{N}$ y devuelve hasta 15 IDs por bundle.

\subsection{Representacion y retrieval}

El backend principal usa OpenCLIP Marqo Fashion SigLIP~\cite{radford2021clip,zhai2023siglip}. La similitud se calcula por coseno sobre embeddings normalizados:
\begin{equation}
\mathrm{sim}(\mathbf{q}, \mathbf{P}) = \mathbf{q}\mathbf{P}^{\top}
\end{equation}

\subsection{Deteccion por regiones}

Cuando se habilita deteccion, se usan cajas de ropa con YOLO~\cite{jocher2023yolo}, con cache reutilizable en JSON. Si no hay detecciones validas, se usa fallback a imagen completa.

\subsection{Reranking}

El pipeline incluye capas opcionales de post-procesado:
\begin{itemize}
  \item filtros de genero/categoria/score;
  \item ajuste por timestamp;
  \item reranker MLP entrenado en \texttt{src/rerank/train\_mlp.py};
  \item fusion de submissions por RRF en \texttt{src/ensemble\_csv.py}.
\end{itemize}

\subsection{Inventario de ideas implementadas}

El codigo actual incluye las siguientes ideas tecnicas, ya implementadas en scripts:
\begin{itemize}
  \item entrenamiento contrastivo OpenCLIP bundle$\\leftrightarrow$product con soporte AMP, acumulacion de gradiente y mineria de negativos duros (\texttt{src/models/retrieval\_openclip.py});
  \item retrieval guiado por deteccion YOLO con cache de cajas y fallback a imagen completa (\texttt{src/infer.py}, \texttt{src/new\_infer.py});
  \item pipeline per-crop con TTA, agregacion multi-crop y indexado FAISS opcional (\texttt{src/new\_infer.py});
  \item filtrado por seccion + clasificacion zero-shot de categoria por crop + boost semantico (\texttt{src/infer\_phase1.py});
  \item estrategia top-1 por crop para predicciones compactas (\texttt{src/infer\_top1.py});
  \item rerank temporal por proximidad de timestamps extraidos de URL (\texttt{src/new\_infer.py}, \texttt{src/infer\_phase1.py});
  \item reranker MLP listwise con features de pares embedding y mezcla con score base (\texttt{src/rerank/train\_mlp.py}, \texttt{src/infer\_phase1.py});
  \item penalizacion de hubness y rerank con modelo pesado de vision (late interaction) (\texttt{src/reranker.py});
  \item ensemble final de predicciones por Reciprocal Rank Fusion (\texttt{src/ensemble\_csv.py});
  \item utilidades de datos reproducibles: preprocesado robusto, augmentacion offline determinista, inferencia de genero y analitica temporal (\texttt{src/utils/}).
\end{itemize}
