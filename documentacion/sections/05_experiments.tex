\section{Protocolo Experimental}

\subsection{Estrategia de Evaluacion Local}

La evaluacion local se ejecuta en inferencia con split por \texttt{bundle\_asset\_id} controlado por \texttt{infer.val\_ratio}. Segun entrypoint, se reportan \texttt{Hit@K}/\texttt{Recall@K} para \texttt{infer.eval\_ks}:
\begin{enumerate}
  \item \texttt{src.infer} (pipeline principal).
  \item \texttt{src.new\_infer} (v8).
  \item \texttt{src.infer\_phase1}.
  \item \texttt{src.infer\_top1}.
\end{enumerate}

\subsection{Artefactos de Salida}

Artefactos frecuentes del flujo train/infer:
\begin{itemize}
  \item \texttt{metrics.jsonl}, \texttt{best.pt}, \texttt{epoch\_X.pt} en entrenamiento.
  \item \texttt{test\_submission.csv} y \texttt{val\_metrics.json} para \texttt{src.infer}.
  \item \texttt{test\_submission\_v8.csv}, \texttt{val\_metrics\_v8.json} para \texttt{src.new\_infer}.
  \item \texttt{test\_submission\_phase1.csv}, \texttt{val\_metrics\_phase1.json} para \texttt{src.infer\_phase1}.
  \item \texttt{test\_submission\_top1.csv}, \texttt{val\_metrics\_top1.json} para \texttt{src.infer\_top1}.
\end{itemize}

\subsection{Reportes Analiticos Observados en el Workspace}

\texttt{outputs/ts\_date\_alignment\_report.csv} (1,876 bundles):
\begin{itemize}
  \item 6,493 productos enlazados con timestamp disponible.
  \item 12.26\% de productos comparten fecha exacta bundle-producto.
  \item 54.57\% comparten mes.
  \item 81.01\% comparten trimestre.
\end{itemize}

\texttt{outputs/category\_section\_summary.json}:
\begin{itemize}
  \item 88 categorias totales en joins train.
  \item Para categorias con al menos 30 muestras: 9 son single-section y 10 casi single-section (umbral de pureza 0.95).
\end{itemize}
