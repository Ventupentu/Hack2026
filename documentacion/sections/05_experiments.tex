\section{Plan Experimental (Sin Resultados)}

\subsection{Objetivo del plan}

Esta seccion define el protocolo de experimentacion para comparar variantes de pipeline sin reportar metricas finales. El foco es describir \textit{que} se prueba y \textit{como} se ejecuta cada prueba.

\subsection{Protocolo operativo}

\begin{enumerate}
  \item Fijar semilla y configurar run con Hydra.
  \item Codificar indice de productos para la variante seleccionada.
  \item Ejecutar inferencia sobre split de validacion local controlado por \texttt{infer.val\_ratio}.
  \item Exportar artefactos de run para trazabilidad y comparacion posterior.
\end{enumerate}

\subsection{Artefactos esperados por run}

\begin{itemize}
  \item \texttt{metrics.jsonl}, \texttt{best.pt}, \texttt{epoch\_X.pt} en entrenamiento.
  \item \texttt{test\_submission*.csv} y \texttt{val\_metrics*.json} en inferencia.
  \item Caches de deteccion en JSON bajo \texttt{data/yolo\_detections} u otra ruta configurada.
\end{itemize}
