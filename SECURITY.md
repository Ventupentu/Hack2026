# Security Policy

## Supported Versions

This project does not currently maintain multiple release branches.
Security fixes are applied to the latest version in the main branch.

## Reporting a Vulnerability

If you find a security vulnerability, please report it privately.

Preferred method:
- Open a private GitHub Security Advisory in this repository.

Alternative method:
- Contact the maintainers directly if private advisory is not available.

Please include:
- A clear description of the issue
- Steps to reproduce
- Potential impact
- Any suggested fix (if available)

## Response Process

We will acknowledge reports as soon as possible and work on a fix based on severity.
Please avoid public disclosure until a fix is available.
